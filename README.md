# 🏡 California Housing Prices – Análisis y Modelado Predictivo

Proyecto de **análisis y predicción de precios de viviendas en California** usando datos reales. Combina **EDA**, limpieza de datos y **modelos predictivos** para identificar patrones y generar insights accionables.

---

## Descripción del Proyecto

- **Análisis Exploratorio de Datos (EDA)**: exploración de distribuciones, correlaciones y tendencias geográficas.  
- **Limpieza de Datos**: detección y tratamiento de **outliers** y valores faltantes para asegurar calidad.  
- **Feature Engineering**: creación de variables que mejoran el rendimiento de los modelos.  
- **Modelos Predictivos**: Regresión Lineal y Random Forest para estimar precios de viviendas.  
- **Visualización de Datos**: gráficos y mapas para apoyar decisiones basadas en datos a nivel operativo y directivo.

---

## Insights Clave

- Los precios varían significativamente según **ubicación geográfica** y características demográficas.  
- Algunos **indicadores económicos** impactan fuertemente el valor medio de las viviendas.  
- La limpieza de datos y selección de variables mejora la **precisión y confiabilidad** de los modelos.
